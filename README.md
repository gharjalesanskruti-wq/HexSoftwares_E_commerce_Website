# 👜 StyleOra - Luxury Handbag E-Commerce Website UI/UX Design

A modern luxury handbag e-commerce website designed in Figma focusing on elegant product presentation, intuitive navigation, and a seamless shopping experience.


## 📌 Project Overview

This project was created to explore and implement modern E-commerce UI/UX principles including:

- Product Categorization
- Product Showcase
- Customer Reviews
- Shopping Cart Experience
- Checkout Flow
- Responsive Layout Design
- User-Centered Navigation
The design focuses on providing customers with a premium shopping experience while maintaining simplicity and usability.


## 🎯 Objective

Learn about e-commerce principles, product categorization, and checkout processes.

Design a website with:

- Clear Product Showcase
- Easy Navigation
- User-Friendly Interface
- Seamless Checkout Experience
- Premium Visual Design

## 🧠 UI/UX Principles Applied

### Visual Hierarchy
Designed clear content flow to guide users toward key actions.

### Consistency

### Accessibility
Used readable typography and high-contrast elements.

### User-Centered Design
Focused on minimizing friction during product browsing and purchasing.
